Background

The WebXR API allows people to experience 3-D Virtual Reality(VR) and Augemented Reality(AR) using their computers, mobile devices, and reality headsets! It will be one of the topics (along with React) in the CS77A course of the proposed Web Application Developer certificates of achievements.

What you need for today:

VR Headsets, mobile devices, laptops and desktops are all supported as long as they have a modern Web browser such as Chrome, Safari, Firefox, the Quest web browser, or Wolvic (and reportedly Opera should work too). VR headsets using the Quest web browser or Wolvic (both are free apps) will have the best experience.
  1. Make sure you are using this site using https as some VR features are not allowed when using http. Use this link: https://webxrexpo.glitch.me/
  2. How to Enable VR Audio and Control Movement is a must-read, especially if you are using Chrome.
